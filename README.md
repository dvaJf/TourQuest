# Travel App

## Описание
Travel App - это веб-приложение для прохождения туров с использованием QR-кодов. Пользователи могут регистрироваться, проходить туры, сканировать QR-коды на местах и получать награды.

## Технологии
- Python 3.11
- Django 6.0.3
- SQLite3 (база данных)
- Pillow (обработка изображений)

#### Установка и запуск
1. Склонируйте репозиторий
2. Перейдите в директорию проекта: `cd trav`
3. Соберите и запустите контейнер: `docker-compose up --build`
4. Откройте браузер и перейдите по адресу: http://localhost:8000

#### Команды для работы с Docker
- `docker-compose up` - запуск приложения
- `docker-compose up --build` - сборка и запуск приложения
- `docker-compose down` - остановка приложения
- `docker-compose exec web python manage.py <команда>` - выполнение Django команд (например, создание суперпользователя)

## Структура проекта
- `travel/` - основной Django-проект
- `main/` - приложение с основной функциональностью
- `media/` - загрузка медиафайлов (аватары, изображения туров)
- `db.sqlite3` - файл базы данных

- Магазин внутри приложения
